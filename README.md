# Breathe — Releases

Official releases for **Breathe** client — a multi-protocol VPN with adaptive traffic masking.

## Downloads

| Platform | File | Description |
|----------|------|-------------|
| **Windows** | `Breathe_Setup_X.X.X.exe` | Windows installer (requires admin) |
| **Android** | `Breathe_vX.X.X.apk` | Android APK (sideload) |

## How to Install

### Windows
1. Download `Breathe_Setup_X.X.X.exe` from [latest release](../../releases/latest)
2. Run the installer (may require admin rights)
3. Launch Breathe VPN from desktop shortcut
4. Enter your invite code

### Android
1. Download `Breathe_vX.X.X.apk` from [latest release](../../releases/latest)
2. Allow installation from unknown sources
3. Install and open the app
4. Enter your invite code

## Auto-Update

The app checks for updates automatically on startup by reading `version.json` from this repository.

## Features

- 🔐 Multi-protocol cascade
- 🎭 Adaptive traffic masking
- 🖥️ Windows & Android support
- 🔀 Split Tunneling (per-app on Android, per-process on Windows)
- 🏠 Self-hosted — your server, your rules
